# Capacité : Analyse Métier

## Objectif
Analyser une demande utilisateur brute pour en extraire les informations structurantes nécessaires au projet.

## Méthode d'Analyse (Entonnoir)

### 1. Identifier le "Pourquoi" (Objectif)
- Quel est le but ultime du site ? (Vendre, Informer, Convaincre ?)
- Quel problème résout-il ?

### 2. Identifier le "Pour Qui" (Cible)
- Qui est l'utilisateur final ? (Clients, Élèves, Partenaires ?)
- Quelle est sa maturité numérique ?

### 3. Identifier le "Quoi" (Périmètre)
- Quelles sont les fonctionnalités clés ? (Contact mailto, Portfolio, Galerie ?)
- Quel est le volume de contenu estimé ?

## Format de Sortie (Interne)
L'analyse doit permettre de remplir les sections du template `templates/cahier-des-charges-template.md` sans ambiguïté.
