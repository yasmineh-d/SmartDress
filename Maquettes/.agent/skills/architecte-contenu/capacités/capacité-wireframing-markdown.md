# Capacité : Wireframing Markdown

## Objectif
Produire un schéma structurel de page lisible par un humain ET une machine.

## Format Standard
```markdown
# Wireframe : [Nom de la Page]

## ZONE 1 : [Nom Zone] (Header/Hero/...)
- **Composant** : [Type] (Menu, Bouton, Titre)
- **Contenu** : "[Texte approximatif]"
- **Action** : Vers [Lien]

## ZONE 2 : ...
```
