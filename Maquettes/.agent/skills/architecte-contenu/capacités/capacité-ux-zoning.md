# Capacité : UX Zoning

## Objectif
Définir les zones fonctionnelles d'une page avant de détailler le contenu.

## Les Zones Standards
1. **Header** : Navigation, Identité, CTA principal.
2. **Hero** : Proposition de valeur unique (UV), Visuel clé.
3. **Body** : Contenu principal, Preuve sociale, Arguments.
4. **Footer** : Navigation secondaire, Crédits, Légal.

## Règles Ergonomiques
- Le CTA principal doit être visible "above the fold" (au dessus de la ligne de flottaison).
- Les formulaires doivent être courts.
- La hiérarchie visuelle doit guider l'œil en Z ou en F.
