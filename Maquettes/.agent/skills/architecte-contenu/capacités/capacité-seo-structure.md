# Capacité : SEO Structure

## Objectif
Créer une architecture de site (Sitemap) logique et performante.

## Règles d'Arborescence
1. **Règle des 3 clics** : Aucune page ne doit être à plus de 3 clics de l'accueil.
2. **Siloing** : Regrouper les pages par thématique sémantique.
3. **Menu Principal** : Limiter à 5-7 entrées maximum.

## Template Sitemap
- [Accueil] (Home)
    - [Service A]
        - [Sous-service A1]
    - [Service B]
    - [Contact]
